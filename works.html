MoveGUI by Hand
(2024.03 - 2025.01)
高校時代に課題研究で制作。MediaPipeを用いたハンドジェスチャ認識によって、上下スクロールやクリック操作のような画面操作をWebカメラを通して非接触で可能としたアプリケーション。

【技術スタック】
Frontend: Tkinter
Backend: Python (MediaPipe, scikit-learn, NumPy, PyAutoGUI, OpenCV, Joblib)
Infrastructure: Raspberry Pi 4 Model B 4GB, RasTechカメラモジュール

【GitHub】 非公開
gakushoku-bot
(2025.11)
毎日定時に会津大学の学食メニューをDiscordへ通知するアプリケーション。
10:00に『日替わりランチ』、15:30に『夕食セット』、19:00に『明日のメニュー予告』を通知する。

【技術スタック】
Frontend: Discord Client
Backend: Discord Webhook API, JavaScript
Infrastructure: Cloudflare Workers

【GitHub】 https://github.com/yonu33/gakushoku-bot
uoa-hub-auth-bot
(2025.11)
大学コミュニティや関連団体のDiscordサーバー参加者向けのメール認証システム。
大学ドメインのメールアドレスで認証し、自動でロールを付与する。 解説記事

【技術スタック】
Frontend: Discord Client, HTML, CSS
Backend: JavaScript (Discord API, Gmail API)
Database: Google Sheets
Infrastructure: Cloudflare Workers, Google Apps Script

【GitHub】 非公開
